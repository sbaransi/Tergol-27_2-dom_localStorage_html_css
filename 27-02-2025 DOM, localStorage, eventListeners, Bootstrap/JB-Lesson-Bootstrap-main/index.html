Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

  • Container: Centers content and provides padding
  • Rows: Horizontal groups of columns that ensure proper alignment
  • Columns: Content containers with different width options
  • Breakpoints: xs (<576px), sm (≥576px), md (≥768px), lg (≥992px), xl (≥1200px), xxl (≥1400px)
